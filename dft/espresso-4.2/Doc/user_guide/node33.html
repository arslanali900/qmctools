7 Post-processing

There are a number of auxiliary codes performing postprocessing tasks such as plotting, averaging, and so on, on the various quantities calculated by .x. Such quantities are saved by .x into the output data file(s). Postprocessing codes are in the PP/ directory. All codes for which input documentation is not explicitly mentioned have documentation in the header of the fortran sources.
